■ Logo

Logo usage

The Fastn mark + wordmark. Use the wordmark as the primary logo. Icon mark alone only where the brand is already established.

Wordmark — light background
Primary usage on light surfaces
Wordmark — dark background
Inverted for dark surfaces
Icon mark — rounded
App icons, favicons, social avatars
Icon mark — square
Where rounded corners are clipped
LinkedIn profile
Sized for LinkedIn profile image

Logo rules

Do

  • Use the official logo files as provided
  • Keep minimum clear space around the logo
  • Use white logo on dark, black logo on light
  • Scale proportionally

Don't

  • Stretch, rotate, or skew the logo
  • Add drop shadows, outlines, or effects
  • Place on busy backgrounds
  • Recolor the logo (brand purple is the icon, not the wordmark)

■ Voice & Tone

How Fastn speaks

Clear, confident, developer-first. Every word earns its place.

★ Principles
  • Clear over clever
  • Confident, not arrogant
  • Technical precision with warmth
  • Developer-first, jargon-aware
✎ Writing rules
  • Use active voice
  • Lead with the benefit
  • Short sentences. Short paragraphs.
  • Talk to the reader — use "you"
  • Code examples > abstract explanations
  • Sentence case for headings

Tone by context

Marketing
Bold, energetic, forward-looking
"Connect your entire stack in minutes. No code required."
Documentation
Precise, patient, example-driven
"Pass your API key in the Authorization header. Here's how:"
Error messages
Calm, actionable, no blame
"Connection timed out. Check your API key and try again."
Product UI
Direct, helpful, quietly confident
"3 integrations connected. 12 events synced today."

Vocabulary

Prefer

  • connect, integrate, automate
  • ship, build, fast, simple
  • powerful, reliable, flexible
  • your stack, your workflow

Avoid

  • leverage, synergy, disrupt
  • revolutionary, game-changing
  • seamless, cutting-edge
  • best-in-class, world-class
